Problèmes et défis

Le script de changement de personnage a été un défi à créer. Le premier problème rencontré était comment trouver les joueurs. Pour régler le problème j'ai décidé de donner les Game Objects en référence et de vérifier combien de joueurs sont assigné pour ne pas causer d'erreurs.

Le deuxième problème a été de faire marcher le script peut-importe quel joueur est jouable. Au début le script ne fonctionnait qu'avec deux joueurs, mais lorsque j'ai ajouté un autre joueur j'ai dû revoir la manière que j'avais fait le script. Le script se trouve sur un Game Object vide qui contrôle le changement de personnage

Projet Unity

Projet réalisé dans le cadre du cours de jeu vidéo 2. Le jeu est un "First Person Shooter" où l'on doit éliminer des robots zombies envahissant une ville que l'on peut nommer. On peut récupérer des objets et les déposer dans une zone de collection pour améliorer la vitesse de tir.
Réalisé en Hiver 2023 dans le cours de jeu-vidéo 2 au Cégep Édouard-Montpetit. Le projet utilise le logiciel Unity et est programmé en C# avec VS Code. Le projet utilise des fonctionnalités comme le système de particule, le système d’animation, le NavMesh, le système d’audio Unity. Le logiciel github desktop a été utilisé pour l’archivage du projet.

Problèmes et défis

J’ai eu plusieurs problèmes lors de la réalisation de ce projet. Le plus gros d’entre eux était que je n’étais pas capable de tester mon jeu à la maison. J’ai rencontré un problème étrange dont je n’ai jamais su la cause qui empêchait le player input de marcher. Le plus étrange est que le projet marchait parfaitement sur les ordis d’écoles.

L’autre gros problème était que le saut du personnage ne marche pas toujours. Le mesh du terrain que j’ai utilisé n’était pas bien optimisé pour le mouvement du personnage de mon jeu ce qui empêchait le joueur d’être dans l’état “Grounded”.

Le dernier défis rencontré est que nous n’avions pas encore eu de cours de modélisation 3D et que les assets que j’ai trouvés sur le Unity asset store n’était pas exactement les meilleurs, ce qui donne un résultat peu plaisant pour les yeux. Malgré cela je suis quand même fier de mon script de tir que j’ai codé par moi-même avec ce que j’ai appris pendent le cours.

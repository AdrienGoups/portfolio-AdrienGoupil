Projet Unity

Projet réalisé dans le cadre du cours de jeu vidéo 2. Le jeu est un "First Person Shooter" où l'on doit éliminer des robots zombies envahissant une ville que l'on peut nommer. On peut récupérer des objets et les déposer dans une zone de collection pour améliorer la vitesse de tir.
Réalisé en Hiver 2023 dans le cours de jeu-vidéo 2 au Cégep Édouard-Montpetit. Le projet utilise le logiciel Unity et est programmé en C# avec VS Code. Le projet utilise des fonctionnalités comme le système de particule, le système d’animation, le NavMesh, le système d’audio Unity. Le logiciel github desktop a été utilisé pour l’archivage du projet.
